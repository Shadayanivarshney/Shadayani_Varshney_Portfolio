Tracking of Underwater Objects Using Kalman Filter

Description:

This project introduces a system for tracking a detected underwater object using the recorded video. The proposed system implements Kalman Filter by predicting the next object location from the current detected object location and Hungarian Algorithm for tracking the objects.

Movie Recommendation System

Description:

The objective of this movie recommendation system is to provide accurate movie recommendations to users by taking in a movie that a user currently likes as input i.e., by analyzing the contents (storyline, genre, cast, director, keywords etc.) of the movie to find out other movies which have similar content and then ranking similar movies according to their similarity scores, popularity and vote count etc withthe help of machine learning.

Big data and its Visualization

Description:

Big Data when used in accordance with infographics and other visual elements, allows to generate answers faster. Visualization is an important approach to helping Big Data get a complete view of data and discover data values. Here, it is discussed how Big Data analytics and visualization can be integrated to get the best result in Big Data applications.

Failure of Linear Solvers and Corrective Measures

Description:

The increasing availability of advanced-architecture computers has a significant effect on all spheres of scientific computation, including algorithm research and software development in numerical linear algebra. This project gives an overview of widely used numerical methods for the solution of linear systems of equations. The preconditioning approach strikes the balance between the success and failure of a numerical method.
